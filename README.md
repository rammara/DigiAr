# Проект DigiAr

### ExchangeMocker
— имитирует поток данных, поступающих от различных источников.

### Hermes
— реализация на asp.net/MVP, предназначенная для обработки данных и передачи их в базу данных **Mnemosyne**, которая отвечает за хранение и обработку сырых данных и «чеков».

### Mnemosyne
— разработка на asp.net/Web API, взаимодействующая с базой данных Postgres через ORM Entity Framework Core. Отвечает за сохранение данных в базе и выдачу чеков пользователям.
